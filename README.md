# SVG Generator

This project is intended to show an application that can generate simple .svg image files based on user input to decide shape, text color and shape color. It uses the Inquirer Node.js package to collect input from the user, as well as Jest to run unit tests on the code.

## Usage

1. Clone the repository.
2. Install the dependencies:
    ```bash
    npm install
    ```
3. Run the application:
    ```bash
    npm start
    ```
